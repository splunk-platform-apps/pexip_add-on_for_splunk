# Source types for the Pexip Add-On

The Pexip Add-on for Splunk provides the index-time and search-time knowledge for metrics, service status, and service message events in the following formats.

| Sourcetype | Description |
|:---:|---|
| `pexip:status:conference`  | Monitor current active calls and media resource usage |
| `pexip:history:conference` | Historical data on conferences, duration and quality for trend analysis and capacity reporting |
| `pexip:history:participants` | Historical data on ended calls participants |
| `pexip:history:alarms` | Historical data on reported alarms |
