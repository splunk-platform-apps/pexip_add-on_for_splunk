# Upgrade the Pexip Add-on for Splunk

Before releasing 1.0.0 the Pexip Add-on for Splunk is still in the development phase, therefore version 1.0.0 is not backward compatible and will result in complete data duplication due to major checkpoint and events ingestion changes.
